== Description ==

[GraphComment](https://graphcomment.com/en/features.html) is a free & innovative comment plugin for great discussions without Trolls & easy reading. No more disorganized discussions piled on several pages. With GraphComment, your audience can easily follow the relevant discussions. The BubbleRank sort the answers in order of relevance.
We build a dedicated plugin for a seamless integration in WordPress. Insert all comments to the WordPress database. This also allows you to revert to WordPress default commenting without losing any comments that you had while using GraphComment

* Great Responsive web design look-and-feel optimized for Mobile and themeable to adapt to your site.
* Manage directly all your comments into WP-panel with embeded GC-admin
* Import all your old comments from WordPress database
* Auto-sync (backup) of comments from GraphComment to WordPress database
* Be notified in real time for pending comments into WP-panel
* SEO-friendly (structured data) - Comments indexable by search engines (WP version of comments for bots)

== Installation ==

**AUTOMATIC INSTALLATION**

1. Click on PLUGINS > Add New

2. Search for "GraphComment"

3. Click on Install Now (FTP ID Required)

4. Activate Plugin

5. From your blog administration (WP Admin panel), click on GraphComment menu to change settings

**MANUAL INSTALLATION**

1. Download GraphComment plugin and unpack archive to the 'wp-content/plugins/' directory inside
   of WordPress

  * Maintain the directory structure of the archive (all extracted files
    should exist in 'wp-content/plugins/graphcomment-comment-system/'
  * Deactivate any third party comment plugins you may have previously installed from the Plugins menu.

2. Activate Plugin

3. From your blog administration (WP Admin panel), click on GraphComment menu to change settings

**SETTING UP THE PLUGIN**

1. Click on CREATE AN ACCOUNT button to add your website and create your GraphComment Website ID.
2. Activate the plugin from the date of your choice or always.
3. Activate the synchronization to auto backup to WordPress database if you want. (need to sign in)

== Frequently Asked Questions ==

= Is GraphComment free to use on my site? =

Yes, GraphComment is free to use. All basic features of GraphComment are totally free of charge with «PERSONAL» plan. With this offer, all GraphComment basic features are included and the system is totally functional. If you happen to need more features, you will be invited to purchase the desired feature. Please refer to [pricing page](https://graphcomment.com/en/pricing.html) for more information .

= How do I customize the look-and-feel of GraphComment ? =

You can choose a theme color on an elegant free selection to nicely integrate GraphComment in your webdesign.

= Will I lose comments if I deactivate GraphComment ? =

You are able to synchronize all new GraphComment comments into the existing WordPress data blog. You will then be able to have a back up copy of all comments. Then you will never loose comments if you deactivate GraphComment.
Go to the tab “Synchronization”. Then syncs will start, and repeat multiple time per day.

= How do I keep the old comments of my WordPress blog ? =

When installing, you will be able to select the date of start of the GraphComment comments. All previous comments will be shown in the ancient way. ie: if you select the date of installation of GraphComment, then you will be able to see all ancient comments as well.

= How do I set up Single Sign-On (SSO)? =

SSO allows users in your database to comment without requiring them to register with GraphComment. Access to SSO is currently available as a SDK for users with a [Paid plan](https://graphcomment.com/en/pricing.html). If you would like to enable this feature, [contact our Support team](https://support.graphcomment.com/en/contact-us/). Also check out our Github documentation for [setting up SSO on WordPress](https://github.com/graphcomment/sdk-api-php).

= Do I still remain the owner of the data when using GraphComment ? =

Yes, you own your data. Site owners can easily retrieve all comment data of their site at any time.
Users own their content and provide a permanent license to GraphComment having accepted our terms and conditions. This includes the management of any issues of liability that may arise with this content. For the media site owner, we also provide them a permanent license. This means that he can recover all the content produced on his website, and that he can dispose of and use, as he sees fit and in permanent way. For this we put at his disposal a tool to export all data in the administration of his website.
